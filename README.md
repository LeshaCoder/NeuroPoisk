# НейроПоиск — умный поиск по книгам

Десктопное приложение на Python с графическим интерфейсом. Загружаешь книгу или папку с книгами — и можешь искать нужные фрагменты или задавать вопросы по тексту на русском языке.

---

## Описание решения и стек технологий

Приложение реализует подход **RAG (Retrieval-Augmented Generation)**:

1. Текст книги разбивается на фрагменты и индексируется с помощью векторных эмбеддингов локально на компьютере
2. По запросу система находит наиболее смыслово близкие фрагменты (не просто по совпадению слов, а по смыслу)
3. Найденные фрагменты передаются в языковую модель, которая формирует ответ **строго по тексту книги**

**Стек технологий:**

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.10+ |
| Интерфейс | `customtkinter` (десктоп, Windows) |
| Векторные эмбеддинги | `sentence-transformers`, модель `all-mpnet-base-v2` |
| Векторная база данных | `piragi` (на основе LanceDB) |
| Языковая модель | OpenRouter API — бесплатные LLM |
| Чтение PDF | `pymupdf` |
| Чтение DOCX | `python-docx` |
| Чтение EPUB / FB2 / MOBI | `ebooklib`, встроенные парсеры |
| Сборка в .exe | `PyInstaller` |

## Как запустить сервис

### Вариант 1 — готовый `.exe` (рекомендуется)

1. Скачай `NeuroPoiskov.exe` из папки `dist/`
2. Запусти двойным кликом — установка не требуется
3. Нужен доступ в интернет для получения ответов от языковой модели

### Вариант 2 — из исходников

Требования: **Python 3.10+**, Windows 10/11

```bash
# 1. Установить зависимости
pip install customtkinter piragi sentence-transformers pymupdf python-docx chardet ebooklib beautifulsoup4 lxml

# 2. Запустить
python neuropoisk_app.py
```

### Сборка `.exe` самостоятельно

```bash
pyinstaller --onefile --windowed --name "NeuroPoiskov" --icon "icon.ico" --add-data "icon.ico;." --hidden-import=chardet --hidden-import=ebooklib --hidden-import=bs4 --hidden-import=lxml --hidden-import=fitz --hidden-import=docx --hidden-import=sentence_transformers --hidden-import=piragi --hidden-import=customtkinter --collect-all customtkinter --collect-all sentence_transformers --collect-all piragi neuropoisk_app.py
```

## Как загрузить книги

В интерфейсе есть две кнопки в верхней части:

- **`📁 Папка`** — выбрать папку, программа найдёт все поддерживаемые файлы внутри
- **`📄 Файл`** — выбрать один файл

После выбора программа автоматически читает и индексирует текст. Прогресс отображается в реальном времени:

```
📖  Читаю файл...  62%
⏱  14 сек  |  осталось ~9 сек
```

Когда готово — появляется сообщение и можно задавать вопросы:

```
✅  Файл прочитан и готов к работе.
   Введите вопрос и нажмите «Спросить».
```

**Поддерживаемые форматы:** `.txt`, `.fb2`, `.epub`, `.mobi`, `.azw3`, `.pdf`, `.docx`, `.rtf`, `.odt`, `.html`, `.htm`, `.djvu`, `.md`, `.csv`

> Файл `.txt` — основной формат по условию задания, поддерживается в полной мере.

## Примеры работы

---

### Пример 1 — поиск фрагментов

> Тестовая книга: **«Мастер и Маргарита»** М. А. Булгакова, файл `master_i_margarita.txt`

**Запрос:**
```
Найди где говорится про допрос Иешуа
```

**Результат:** система возвращает 3–5 фрагментов с указанием источника

```
📄  Фрагмент 1  —  master_i_margarita.txt
— Ты — великий философ? — иронически осведомился прокуратор.
— Нет, — ответил арестант, — я не философ. Я говорю тебе
правду, и если тебя это беспокоит, лучше отпусти меня...

📄  Фрагмент 2  —  master_i_margarita.txt
Пилат вскинул глаза на арестанта и увидел, что рядом с тем
вспыхнуло солнце... Игемон поднял голову, чтобы взглянуть
прямо в лицо арестанту...

📄  Фрагмент 3  —  master_i_margarita.txt
— Откуда ты родом? — И ты не знаешь, откуда я? Из города
Гамалы, — ответил арестант, показывая головой, что там,
направо, где-то далеко, за садом, есть город Гамала...
```

### Пример 2 — ответ на вопрос с цитатами

> Тестовая книга: **«Преступление и наказание»** Ф. М. Достоевского, файл `prestuplenie_i_nakazanie.txt`

**Запрос:**
```
Почему Раскольников решился на убийство?
```

**Результат:** развёрнутый ответ по тексту + список цитат-источников

```
Раскольников совершил убийство, опираясь на собственную теорию
о «праве» исключительных людей преступать нравственный закон.
Он делил людей на «обыкновенных» — тех, кто живёт в покорности,
и «необыкновенных» — тех, кому позволено «переступить» ради
высшей цели. Убийство старухи-процентщицы было для него
проверкой: способен ли он на поступок или он «тварь дрожащая».

📎  Источники:
  • «Тварь ли я дрожащая или право имею...»  [prestuplenie_i_nakazanie.txt]
  • «...я хотел Наполеоном сделаться, оттого и убил...»  [prestuplenie_i_nakazanie.txt]
  • «Я не старуху убил — я себя убил...»  [prestuplenie_i_nakazanie.txt]
```

## Частые проблемы

**Долгая индексация** — нормально для больших книг (300+ страниц). Прогресс и оставшееся время видны на экране.

**«Lance не работает на exFAT/FAT32»** — книга лежит на флешке. Скопируй файл на локальный диск (C: или D:).

**«Все модели недоступны»** — проверь интернет. Бесплатные модели OpenRouter иногда временно перегружены — подожди минуту и попробуй снова.
