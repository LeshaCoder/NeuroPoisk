# НейроПоиск

Программа для поиска по документам с помощью искусственного интеллекта. Загрузи файл или папку — и задавай вопросы на русском языке.

---

## Быстрый старт

1. Запусти `NeuroPoiskov.exe`
2. Нажми **«Файл»** или **«Папка»** и выбери документ
3. Дождись надписи **«Файл прочитан и готов к работе»**
4. Введи вопрос и нажми **«Спросить»** или **Ctrl+Enter**

---

## Поддерживаемые форматы

| Тип | Форматы |
|-----|---------|
| Электронные книги | `.fb2`, `.epub`, `.mobi`, `.azw3`, `.azw` |
| Документы | `.pdf`, `.docx`, `.rtf`, `.odt` |
| Текст | `.txt`, `.md`, `.csv`, `.html`, `.htm` |
| Сканы | `.djvu`, `.djv` |

Можно загрузить как один файл, так и целую папку — программа обработает все документы внутри.

---

## Требования

- Windows 10 / 11
- Интернет-соединение (для получения ответов через OpenRouter)
- Для `.djvu` файлов — установленный [DjVuLibre](https://djvu.sourceforge.net/)
- Для `.epub` / `.mobi` — библиотеки устанавливаются автоматически вместе с программой

---

## Где хранится индекс

При первом открытии файла программа создаёт индекс в папке:

```
%TEMP%\NeuroPoiskov\
```

Индекс удаляется при закрытии программы. Если файл открывали раньше — повторная индексация всё равно происходит заново.

> **Важно:** программа не работает с документами на флешках формата exFAT / FAT32 и сетевых дисках — индекс записывается только в системный `%TEMP%`.

## Сборка из исходников

Для сборки `.exe` нужен Python 3.10+ и установленные зависимости.

**Установка зависимостей:**
```bash
pip install customtkinter piragi sentence-transformers pymupdf python-docx chardet ebooklib beautifulsoup4 lxml
```

**Сборка:**
```bash
pyinstaller --onefile --windowed --name "NeuroPoiskov" --icon "icon.ico" --add-data "icon.ico;." --hidden-import=chardet --hidden-import=ebooklib --hidden-import=bs4 --hidden-import=lxml --hidden-import=fitz --hidden-import=docx --hidden-import=sentence_transformers --hidden-import=piragi --hidden-import=customtkinter --collect-all customtkinter --collect-all sentence_transformers --collect-all piragi neuropoisk_app1.py
```

Готовый `.exe` появится в папке `dist\`.

## Частые проблемы

**Программа долго читает файл** — это нормально для больших книг. Прогресс и оставшееся время отображаются в интерфейсе.

**«Ошибка индексации: Lance не работает...»** — документ находится на флешке или сетевом диске. Скопируй файл на локальный диск (C: или D:) и попробуй снова.

**Нет ответа / «Все модели недоступны»** — проверь интернет-соединение. Программа использует бесплатные модели OpenRouter, иногда они временно перегружены — попробуй через минуту.

**Иконка не отображается в панели задач** — убедись что файл `icon.ico` лежит в той же папке что и `NeuroPoiskov.exe`.
